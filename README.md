## Функционал

**Подсказка в трее**  
При наведении на иконку показывается краткая сводка: загрузка CPU и температура, ядра (до 8), GPU (nvidia-smi), ОЗУ, диски, скорость сети. Состав строк и интервал обновления (2 с — 10 мин) настраиваются.

**Меню по ПКМ / клику по иконке:**
- **Показать показатели** — открывает окно с полным списком показателей (все ядра, батарея, время работы ПК, swap, число процессов, пинг, суммарный трафик) и настройками.
- **Включить / Выключить оверлей** — выводит те же показатели поверх экрана (угол, цвет текста, подложка настраиваются в окне показателей).
- **Обновить** — принудительное обновление кэша и подсказки.
- **Включить / Остановить запись** — пишет снимки статистики в `tray_stats_log.txt` с интервалом подсказки.
- **Выключить** — полный выход из приложения.

**Окно «Показатели системы»**  
- Полный список показателей (обновление раз в 2 с).  
- Настройки: интервал подсказки, какие строки показывать в подсказке/оверлее, автозапуск с Windows, параметры оверлея (цвет, положение, подложка, жирный шрифт).  
- Кнопки: **Обновить**, **Включить/Остановить запись**, **Выключить** (выход), **Закрыть** (только окно).
- Дополнительно: **Выбор темы** (По умолчанию системная, отдельно темная\светлая), **Выбор языка** (Русский, Белорусский, Татарский, Китайский, Английский)

**Оверлей**  
Текст статистики поверх рабочих окон и поверх приложений в режиме borderless fullscreen. Окно «всегда сверху», клики проходят сквозь него. В эксклюзивном полноэкранном режиме (игры) поверх вывести нельзя — ограничение Windows.

## Установка и запуск

```bash
pip install -r requirements.txt
python run.py
```

Иконка появляется в трее (на Windows 11 может быть в области «^» у часов). Если иконки нет — запустите из терминала или через `run.bat`, в консоли будет видна ошибка.

---

## Сборка в exe

Положите `tray_stats.ico` в корень проекта (рядом с `run.py`).

```bash
pyinstaller tray_stats.spec
```

Готовый exe: `dist/tray_stats.exe`.

### Температура CPU (Windows)

Часто без доп. ПО температура не отдаётся. Варианты:
- **Libre Hardware Monitor** — запустить, свернуть в трей; чтение через WMI.
- **HWiNFO64** — включить общий доступ к датчикам.
- **MSI Afterburner** — температура берётся из общей памяти MAHM, если он запущен.

### Полноэкранные приложения

- Оверлей виден поверх обычных окон и **borderless fullscreen**. В **эксклюзивном полноэкранном** режиме (DirectX/OpenGL) поверх картинки вывести нельзя.
- В полноэкранном режиме панель задач (и трей) часто скрыта — нажмите Win или переведите игру в «оконный полноэкранный» / «без рамки».
